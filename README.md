# Trezor.Net

## Getting Started

- Clone the repo and open the solution
- There is a sample/unit test app there called Trezor.Net.Sample
- This sample is a .NET Core console app and shows you how to get addresses

## NuGet

Install-Package Trezor.Net

## Suported Platforms

- .NET Framework
- .NET Core
- Android
- UWP
